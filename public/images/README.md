# Thư mục chứa ảnh

Đặt các ảnh sau vào thư mục này:

1. `background.png` - Ảnh background chính của trang
2. `form-bg.png` - Ảnh background cho khung nhập liệu

## Cấu trúc thư mục:
```
public/
├── images/
│   ├── background.png     # Ảnh background chính
│   ├── form-bg.png       # Ảnh background cho form
│   └── README.md         # File này
├── index.html
└── ...
```

## Định dạng ảnh được hỗ trợ:
- JPG/JPEG
- PNG  
- WebP
- SVG

## Kích thước khuyến nghị:
- Background: 1920x1080px hoặc lớn hơn
- Form background: 600x400px hoặc tương tự
